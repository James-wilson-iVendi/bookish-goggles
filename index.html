Front-End Tooling for the Webshops Internal Admin UI

Disclaimer

  • None of these tools will make you a better developer...probably.
  • Different projects require different tools.
  • I'll probably change my mind about half this stuff in 6 months time.

Snowpack

“Snowpack is a lightning-fast frontend build tool, designed for the modern web.”

Snowpack looks after

  • Dev server
  • Code compilation
    • JSX
    • TypeScript
    • Sass Modules
  • Build pipeline

Pros

  • Fast startup
  • Fast Refresh
  • Lightweight
  • Decent plugin ecosystem

Cons

  • Doesn't like it when files are renamed or removed
  • Not as a mature as others e.g. WebPack
  • Bundling a bit confusing at first

Redux Toolkit

“The official, opinionated, batteries-included toolset for efficient Redux development”

Pros

  • Reduces boilerplate
  • Immer for working with immutable state
  • Sensible middleware included
  • Good TypeScript support

Cons

  • Opinionated
  • Still just Redux i.e. if you fundamentally dislike Redux this probably won't change that.

React Bootstrap

“The most popular front-end framework Rebuilt for React.”

We also use...

  • react-select
  • react-toastify
  • react-router

Pros

  • I don't want to try and design the perfect button
  • Good docs...that I don't have to maintain
  • Good test coverage...that I don't have to maintain
  • Lets you focus on the business logic

Cons

  • Bloat
  • You might need something custom
